# BillingService – Подписки и Скидки

Проект на C#, демонстрирующий переработку императивного кода в объектно-ориентированном стиле с использованием современных конструкций C# (switch-выражения, pattern matching, локальные функции, Guard-clauses).

---

## Структура проекта

- `Subscriber.cs` – класс, описывающий подписчика.
- `BillingService.cs` – класс, вычисляющий стоимость подписки с учётом:
  - Статуса подписчика (`Trial`, `Basic`, `Pro`, `Student`)
  - Стажа (Tenure)
  - Количества устройств
  - Региональных налогов
- `Program.cs` – пример инициализации подписчиков и подсчёта их итоговой стоимости.

---

## Ключевые особенности

- **Инкапсуляция**: все поля `Subscriber` закрыты (`private`), доступ через свойства.
- **Switch-выражения и pattern matching** для вычисления скидок и налогов.
- **Guard-clauses** для проверки аргументов и null.
- **Локальные функции** внутри метода `CalcTotal` для пошагового расчёта.
- **Возврат кортежей** для методов валидации (`Validate`).

---

## Установка
1) Клонируйте репозиторий
2) Откройте проект в VSCode.
3) Убедитесь, что установлен .NET SDK (например, .NET 7).
4) Сборка и запуск:
```bash
dotnet build
dotnet run
```
